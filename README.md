# Steam Download Monitor

Steam Download Monitor — это Python-скрипт для мониторинга загрузки игр в Steam на операционной системе Windows.  
Программа работает в фоне и позволяет отслеживать скорость загрузки, статус загрузки, а также определять, какая именно игра в данный момент скачивается.

Путь установки Steam определяется автоматически через реестр Windows, поэтому скрипт не зависит от того, в какую директорию установлен лаунчер.

---

##  Функциональные возможности

-  Автоматическое определение пути установки Steam через реестр Windows.  
-  Определение текущей скачиваемой игры по файлам  
  `steamapps/appmanifest_*.acf`.  
-  Учёт статуса загрузки:
  - **«Загружается»**, если Steam сообщает об активной загрузке;
  - **«Пауза»**, если активной загрузки нет.  
-  Определение реальной скорости загрузки по входящему сетевому трафику системы.  
-  Вывод информации:
  - сразу после запуска программы;
  - затем каждую минуту;
  - всего 5 измерений (5 минут работы).
